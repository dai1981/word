set

動詞 / 名詞
発音：/set/（セット）
中学生〜 英検4級 CEFR A1 基本語
置く ／ 設定する ／ ひとそろい
📖 主な意味
  • 1 【動詞】置く・据える She set the cup on the table.（カップをテーブルに置いた）
  • 2 【動詞】設定する・合わせる set an alarm（アラームをセットする）、set a goal（目標を定める）。
  • 3 【動詞】（太陽が）沈む The sun sets in the west.（太陽は西に沈む）
  • 4 【名詞】ひとそろい・組 a set of dishes（食器ひとそろい）。
💡 使い方・例文
① ～を準備する・設立する（set up）
機器の設置・会社の設立など幅広く使う頻出句動詞。
They set up a new company last year.
彼らは昨年新しい会社を設立した。
② 設定する（set an alarm / goal）
時刻・目標・規則などを「定める」。
I set my alarm for six o'clock.
私は6時にアラームをセットした。
③ ひとそろい（a set of）
名詞として「組・セット」を表す。
He bought a set of tools.
彼は工具をひとそろい買った。
⚠️ 過去形・過去分詞も set（無変化）。put, cut と同じパターン
⚠️ 「太陽が沈む」は set、「太陽が昇る」は rise。あわせて覚える
🏛 語源・歴史

古英語 settan は「座らせる・置く」を意味し、sit（座る）の使役形にあたる。ゲルマン祖語 *satjaną に由来し、「何かを所定の位置に据える」という基本義を保っている。
