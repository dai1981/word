surname

名詞
発音：/ˈsɜːrneɪm/（サーネイム）
中学生〜 英検準2級 CEFR B1 頻出語
名字 ／ 姓
📖 主な意味
  • 1 【名詞】名字・姓 family name と同じ。Smith や Tanaka など。
💡 使い方・例文
① 名字（surname）
「姓」。
What is your surname?
あなたの名字は何ですか？
② first name との違い
surname（姓）⇔ first name / given name（名）。
My first name is John and my surname is Smith.
私の名はジョン、姓はスミスです。
③ family name との関係
surname = family name = last name。
Please write your surname in capitals.
名字を大文字で書いてください。
⚠️ surname = family name = last name（名字・姓）。first name = given name（名前）
⚠️ sur- は「上に・追加の」。「（名前に)加えられた名＝名字」
🏛 語源・歴史

接頭辞 sur-（上に・追加の）と name（名前）を組み合わせた語。「（個人名に)加えられた名＝名字・姓」を表す。

sur + name
追加の＋名前
