sure

形容詞 / 副詞
発音：/ʃʊər/（シュア）
小学生〜 英検5級 CEFR A1 基本語
確信して ／ 確かな ／ もちろん
📖 主な意味
  • 1 【形容詞】確信して・自信がある be sure of / that ～（～を確信している)。
  • 2 【形容詞】確かな・間違いない a sure way（確実な方法)。
  • 3 【副詞・返答】もちろん・いいよ Sure!（いいですよ)。
💡 使い方・例文
① ～を確信している（be sure of / that）
「～を確信している」は be sure of ～ / that 節。
I'm sure that he will come.
彼はきっと来ると確信している。
② もちろん（Sure!）
依頼への返答で「いいですよ」。
"Can you help me?" "Sure!"
「手伝ってくれる？」「もちろん！」
③ 確かめる（make sure）
「確認する・必ず～する」。
Make sure you lock the door.
必ずドアに鍵をかけてね。
⚠️ 「～を確信している」は be sure of ～ / be sure that ～
⚠️ make sure (that) ～ は「確認する・必ず～する」。重要表現
🏛 語源・歴史

ラテン語 securus（se-＝なしで + cura＝心配＝心配のない・安全な）に由来し、古フランス語 sur を経て英語に入った。secure（安全な）と同じ語源で、「心配ない＝確かな」という発想。
