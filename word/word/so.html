so

副詞 / 接続詞
発音：/soʊ/（ソウ）
小学生〜 英検5級 CEFR A1 基本語
とても ／ そのように ／ だから
📖 主な意味
  • 1 【副詞】とても・非常に 形容詞・副詞を強める。so happy（とても幸せ）。
  • 2 【接続詞】だから・それで 結果を表す。It rained, so we stayed home.
  • 3 【副詞】そのように・そう I think so.（そう思う）。
💡 使い方・例文
① とても（so + 形容詞）
形容詞・副詞を強調する。
This book is so interesting.
この本はとてもおもしろい。
② だから（接続詞）
前の内容の結果を述べる。
I was tired, so I went to bed early.
疲れていたので、早く寝た。
③ so ～ that 構文
「とても～なので…」を表す。
He was so tired that he fell asleep.
彼はとても疲れていたので眠ってしまった。
⚠️ so のあとは形容詞・副詞（so big）、such のあとは（形容詞＋）名詞（such a big dog）
⚠️ 「そう思う／思わない」は I think so. / I don't think so.
🏛 語源・歴史

古英語 swā（そのように・このように）に由来し、ゲルマン祖語 *swa にさかのぼる。「そのように」という基本義から、程度を強める「とても」や結果を表す「だから」へと用法が広がった。
