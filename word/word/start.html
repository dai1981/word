start

動詞 / 名詞
発音：/stɑːrt/（スタート）
小学生〜 英検5級 CEFR A1 基本語
始める ／ 始まる ／ 出発する
📖 主な意味
  • 1 【動詞】始める・始まる begin とほぼ同じ。後ろに to ～ / ～ing。
  • 2 【動詞】出発する・動き出す start for ～（～へ出発する）。
  • 3 【名詞】始まり・出発 from the start（最初から）。
💡 使い方・例文
① ～し始める（start to / ～ing）
後ろは to不定詞でも動名詞でもよい。
It started to rain. / It started raining.
雨が降り始めた。
② 出発する（start for）
「～へ出発する」。
We started for the station early.
私たちは早く駅へ出発した。
③ begin との違い
start と begin はほぼ同じ。start はやや口語的、機械にも使える。
The car won't start.
車のエンジンがかからない。
⚠️ start と begin はほぼ同義。start のほうが口語的で、「（機械が）動き出す」にも使える（begin は使いにくい）
⚠️ 反対語は finish / end（終わる）
🏛 語源・歴史

古英語 styrtan（跳び上がる・突然動く）に由来する。「びくっと動き出す」が基本義で、そこから「動き出す＝始める・出発する」の意味になった。
