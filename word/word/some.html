some

形容詞 / 代名詞
発音：/sʌm/（サム）
小学生〜 英検5級 CEFR A1 基本語
いくつかの ／ いくらかの ／ ある～
📖 主な意味
  • 1 【形容詞】いくつかの・いくらかの 数えられる名詞・数えられない名詞の両方に使える。
  • 2 【代名詞】いくつか・いくらか Some of them agreed.（彼らのうち何人かは賛成した）。
  • 3 【形容詞】ある～・何らかの some day（いつか）。
💡 使い方・例文
① 肯定文で使う（some）
ふつう肯定文で「いくつかの・いくらかの」。
I have some questions.
私はいくつか質問がある。
② 勧誘・依頼の疑問文
「～はいかが？」と勧めるときは疑問文でも some。
Would you like some tea?
お茶はいかがですか？
③ any との違い
否定文・疑問文では普通 any を使う。
I don't have any money.
私はお金を持っていない。
⚠️ some は基本的に肯定文。否定文・疑問文では any を使うのが原則
⚠️ ただし「～はいかが？」と勧めるときや、Yesの答えを期待する疑問文では some を使う
🏛 語源・歴史

古英語 sum（いくらかの・ある）に由来し、ゲルマン祖語 *sumaz にさかのぼる。「（不特定の）ある・いくらかの」という基本義を古くから保っている。
